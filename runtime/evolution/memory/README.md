# memory

本目录用于已确认的进化记忆。  
它不是顶层通用 `memory/`，也不是平行真相源。

## 当前对象

- `lessons/`：确认后的可复用教训
- `strategies/`：确认后的可复用策略

## 边界

- 只收确认后、可回链的内容
- 不收原始聊天、原始 raw 或未确认提议
- 与 owner 文档冲突时，以 owner 文档为准
- lesson / strategy 至少要能回链：
  - `evidence_links`
  - `graduated_from`
  - `owner_target`
  - `last_verified_at`
