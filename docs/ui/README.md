# ui

本目录用于 `V3` 项目里的正式 UI 规则、页面标准和交互约束说明。  
这里只放规范和说明，不放具体实现代码。

## 适合放什么

- 页面层级、导航结构和布局约束
- 组件使用边界、状态反馈和交互规则
- 视觉一致性、文案口径和空态 / 错态说明
- 与验收、测试、review 相关的 UI 判断口径

## 不放什么

- 可直接运行的实现代码
- 与真实界面不一致的过时截图或随手草图
- 只存在于这里、没有正式依据的独有产品结论

## 联动提醒

- 改 UI 规则时，检查 `src/`、`tests/`、相关 workflow、review 和 release 口径是否要同步
- 当前项目没有 UI 时，本目录可以只保留最小入口说明，不强行铺内容
