# me

本目录是给维护者快速阅读的人看层。
它由 AI 负责维护，但不进入 `V3` 默认真相源。

## 适合放什么

- 项目一句话定位
- 当前项目结构总览
- 主要功能或能力清单
- 核心模块 / 核心流程摘要
- 常用查看入口和跳转导航
- 启用时的 `self-memory/` 复盘摘要与给维护者看的提问协作报告

## 不放什么

- 独有规则结论
- 独有流程约束
- 正式计划、review、错误、发布、ADR
- 无法回链到正式文件的独有事实

## 维护规则

- 当 `README.md`、`AGENTS.md`、`PROJECT_RULES.md`、`docs/`、`runtime/`、核心目录结构或功能边界变化时，检查是否要同步本目录
- 本目录只做汇总和导航；正式结论仍回写到真相源
- `self-memory/` 只给维护者复盘使用，不自动进入 AI 侧默认真相源
- 如果本目录与正式文件冲突，以正式文件为准

## 导航

- 模板定位：`../README.md`
- 常驻指导：`../AGENTS.md`
- 目录地图：`../PROJECT_RULES.md`
- 正式规则：`../docs/`
- 正式结果：`../runtime/`
