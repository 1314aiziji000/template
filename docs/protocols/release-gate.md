# release-gate

## 命中条件

- 正式发布
- 构建产物交付
- 部署
- 环境切换
- 面向下游的版本更新

## 最低检查

1. 当前 review 结论明确
2. 当前验证证据明确
3. `Release Card` 已补齐
4. 冒烟检查已明确
5. 回滚路径已明确

## 证据要求

- 当前发布目标
- 当前是否满足发布条件
- 当前冒烟检查与回滚路径
- 剩余风险与后续动作

## 正式落点

- 发布或交付节点：`runtime/releases/`
- 发布暴露出的错误与返工：`runtime/errors/`
- 必要的结构决策：`runtime/adr/`
