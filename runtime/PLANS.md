# Task / Plan

## 使用说明

- 新项目 bootstrap 完成后，也用本文件承接初始化结论
- 单活跃长任务：直接维护 `runtime/PLANS.md`
- 多活跃长任务：升级到 `runtime/plans/active/` 与 `runtime/plans/archive/`
- 中改及以上、高风险抬级、跨会话承接时必须更新计划
- 这是运行对象，不是说明书；写当前这次任务真实要怎么推进
