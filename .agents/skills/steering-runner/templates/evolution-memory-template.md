# 进化记忆模板

建议文件：

- `runtime/evolution/memory/lessons/YYYY-MM-DD-主题.md`
- `runtime/evolution/memory/strategies/主题.md`

建议骨架：

```md
---
id:
created_at:
memory_kind: lesson
title:
applies_to:
do:
  - 
dont:
  - 
evidence_links:
  - runtime/evolution/archive/
graduated_from:
owner_target:
last_verified_at:
---

# 记忆标题

## 适用边界
- 

## 核心结论
- 

## 备注
- 
```

使用要求：

- `memory_kind` 只允许 `lesson` 或 `strategy`
- 只能记录已确认、可回链的内容
